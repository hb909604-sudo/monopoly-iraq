# 🕌 مونوبولي العراق | Monopoly Iraq

لعبة عقارات ويب مستوحاة من "مونوبولي" العالمية، بنسخة عراقية أصيلة: 18 محافظة
عراقية بدلاً من المدن التقليدية، بطاقات فرصة وصندوق مجتمع بطابع محلي، وهوية
بصرية مستوحاة من دجلة والفرات والنخيل والزخارف الرافدينية.

بُنيت اللعبة بالكامل بـ **HTML + CSS + JavaScript (Vite)** بدون أي خادم
خلفي (Backend)، وتُخزَّن كل بيانات اللعبة في **LocalStorage** داخل متصفحك.

---

## ⚠️ ملاحظة مهمة حول "غرف اللعب" (يرجى القراءة قبل البدء)

اللعبة تدعم نظام غرف عبر روابط قابلة للمشاركة (مثل `?room=ABCD12`)، ويعمل هذا
النظام بشكل ممتاز عند فتح الرابط في **عدة تبويبات أو نوافذ على نفس المتصفح
ونفس الجهاز** (تُزامَن الحالة فورياً عبر LocalStorage + Storage Events).

بما أن المشروع **لا يحتوي على خادم مركزي** (حسب المتطلبات)، فإن مشاركة رابط
الغرفة مع شخص يفتحه على **جهاز أو متصفح مختلف** لن تُظهر له نفس اللعبة، لأن
LocalStorage مقصور على المتصفح الذي أُنشئت فيه الغرفة فقط. هذا قيد تقني
أساسي لأي حل "بدون Backend"، وليس خطأ في الكود.

**للعب الحقيقي متعدد الأجهزة لاحقاً**، يمكن ترقية طبقة `src/game/storage.js`
لتستخدم خدمة مزامنة سحابية مجانية مثل Firebase Realtime Database أو Supabase
أو PartyKit، مع إبقاء بقية الكود (المحرك، الواجهة) كما هو تقريباً.

الاستخدام المثالي حالياً: **اللعب الجماعي على نفس الجهاز** (تبويبات متعددة،
أو تمرير الجهاز بين اللاعبين)، أو اختبار اللعبة محلياً بسهولة.

## 🎮 مميزات اللعبة

- 18 محافظة عراقية حقيقية بأسعار وإيجارات وألوان مجموعات مختلفة.
- بطاقات "فرصة" و"صندوق المجتمع" بمحتوى عراقي أصلي.
- نظام سجن كامل (زيارة، دفع غرامة، بطاقة خروج مجاني، 3 محاولات هروب بالدبل).
- جمع 200 دينار عند المرور بنقطة البداية، ضرائب، موقف مجاني بوعاء أموال.
- دعم 2 إلى 6 لاعبين.
- غرف لعب بروابط مشاركة + توليد رمز غرفة عشوائي.
- شاشة انتظار (لوبي) قبل بدء اللعبة، مع عرض اللاعبين المنضمين.
- دردشة داخل الغرفة.
- مؤثرات صوتية (نرد، شراء، دفع، سجن) وموسيقى هادئة اختيارية (تُولَّد
  عبر Web Audio API بدون ملفات خارجية، ويمكن استبدالها بملف MP3 حقيقي).
- حفظ تلقائي للعبة داخل LocalStorage — أعد فتح رابط الغرفة لاحقاً لإكمال اللعبة.
- تصميم متجاوب (Responsive) للموبايل والكمبيوتر، مع نوافذ حوار وتأثيرات ناعمة.

## 💻 التشغيل محلياً

1. تأكد من تثبيت [Node.js](https://nodejs.org) (نسخة 18 أو أحدث).
2. ثبّت الاعتماديات:

   ```bash
   npm install
   ```

3. شغّل خادم التطوير:

   ```bash
   npm run dev
   ```

4. افتح الرابط الذي يظهر في الطرفية (عادة `http://localhost:5173`).

عند فتح الرابط الرئيسي لأول مرة، سيُنشئ التطبيق تلقائياً غرفة جديدة ويعرض
رابط مشاركتها. افتح نفس الرابط في تبويب آخر لتجربة الانضمام كلاعب ثانٍ.

### بناء نسخة الإنتاج

```bash
npm run build
```

ستُنشأ الملفات النهائية داخل مجلد `dist/`. يمكنك معاينتها محلياً عبر:

```bash
npm run preview
```

## 🚀 النشر على GitHub Pages

المشروع يتضمن ملف `.github/workflows/deploy.yml` جاهزاً للنشر التلقائي. الخطوات:

1. في مستودعك على GitHub، اذهب إلى **Settings → Pages**.
2. تحت **Build and deployment**، اختر المصدر **GitHub Actions**.
3. ادفع (push) أي تعديل إلى فرع `main` — سيعمل الـ Workflow تلقائياً،
   ويبني المشروع (`npm run build`) وينشره.
4. بعد اكتمال العملية (تبويب **Actions**)، سيظهر رابط الموقع في
   **Settings → Pages**، بالصيغة:

   ```
   https://USERNAME.github.io/monopoly-iraq/
   ```

### النشر اليدوي (بديل)

إن كنت تفضل عدم استخدام GitHub Actions:

```bash
npm run build
npx gh-pages -d dist
```

(يتطلب تثبيت الحزمة أولاً: `npm install --save-dev gh-pages`)، ثم فعّل
GitHub Pages من **Settings → Pages** واختر الفرع `gh-pages`.

## 🎨 عن الهوية البصرية

- الألوان: رمل الصحراء وطوب الزقورة (طيني/برتقالي محروق)، أزرق دجلة
  والفرات، أخضر النخيل، وذهب التمر — مع لمسات من ألوان العلم العراقي

  لبقية الواجهة (خط عصري واضح القراءة).

  يُستخدم كفاصل بصري بين الأقسام.
